What Data Should Be Public
  1. 1. Proactively release government information online
  2. 2. Reference and build on existing public accountability and access policies
  3. 3. Build on the values, goals and mission of the community and government
  4. 4. Create a public, comprehensive list of all information holdings
  5. 5. Specify methods of determining the prioritization of data release
  6. 6. Stipulate that provisions apply to contractors or quasi-governmental agencies
  7. 7. Appropriately safeguard sensitive information
How to Make Data Public
  1. 8. Mandate data formats for maximal technical access
  2. 9. Provide comprehensive and appropriate formats for varied uses
  3. 10. Remove restrictions for accessing information
  4. 11. Mandate data be explicitly license-free
  5. 12. Charge data-creating agencies with recommending an appropriate citation form
  6. 13. Require publishing metadata
  7. 14. Require publishing data creation processes
  8. 15. Mandate the use of unique identifiers
  9. 16. Require code sharing or publishing open source
  10. 17. Require digitization and distribution of archival materials
  11. 18. Create a central location devoted to data publication and policies
  12. 19. Publish bulk data
  13. 20. Create public APIs for accessing information
  14. 21. Optimize methods of data collection
  15. 22. Mandate ongoing data publication and updates
  16. 23. Create permanent, lasting access to data
How to Implement Policy
  1. 24. Create or appoint oversight authority
  2. 25. Create guidance or other binding regulations for implementation
  3. 26. Incorporate public perspectives into policy implementation
  4. 27. Set appropriately ambitious timelines for implementation
  5. 28. Create processes to ensure data quality
  6. 29. Ensure sufficient funding for implementation
  7. 30. Create or explore potential partnerships
  8. 31. Mandate future review for potential changes to this policy
